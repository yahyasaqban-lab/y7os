# Y7 OS — Asahi Linux (Apple Silicon)

Native ARM64 installer for Apple Silicon Macs running Asahi Linux.

```bash
curl -fsSL https://raw.githubusercontent.com/yahyasaqban-lab/y7os/main/scripts/asahi/install-asahi.sh | sudo bash
```

Includes: Ollama + Metal GPU, Docker ARM64, Open WebUI, Y7 CLI tools, optimized models.
